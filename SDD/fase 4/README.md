# Fase 4 - Implementar

Este directorio contiene la guia de ejecucion de la Fase 4 del enfoque SDD para la app de finanzas.

La finalidad de esta fase es materializar lo definido en Fase 1, Fase 2 y Fase 3 en implementacion real: estructura del proyecto, base de datos, backend, frontend, pruebas y validacion.

## Documentos

- `01_orden_ejecucion_fase4.md`: secuencia recomendada de implementacion.
- `02_base_datos_en_fase4.md`: momento y proceso de creacion real de la base de datos.

## Regla de implementacion

Todo lo correspondiente a implementacion real del producto debe quedar dentro de:

- `app_finanzas/`

La documentacion de soporte y decisiones sigue quedando en:

- `SDD/`
